Slice Tests as a Mermaid chart type

This page registers the Slice Tests diagram as an external Mermaid diagram, then renders a fenced <pre class="mermaid"> block whose body starts with sliceTests. Mermaid routes it to the adapter (slice-tests-mermaid.js) which draws the Given / When / Then test patterns using the same shared d3 drawing module as the standalone renderer.

The source is loaded from blueprint_sliceTests; edit that file and the diagram will auto-refresh.
