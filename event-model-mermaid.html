Event Model as a Mermaid chart type

This page registers the Event Model diagram as an external Mermaid diagram, then renders a fenced <pre class="mermaid"> block whose body starts with eventModel. Mermaid routes it to our adapter (event-model-mermaid.js), which reuses the same parser, layout, and d3 drawing code as the standalone demo.

The source in the mermaid block below:
